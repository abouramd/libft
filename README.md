## Code Details
- Creat the library whit this command.
### This command is to creat the Mandatory part.
  ```
  make
  ```
  or
  ```
  make all
  ```
  or
  ```
  make libft.a
  ```
### to make the Bonus part use this command.
  ```
  make bonus
  ```
